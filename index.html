Farfisa Of The Future

This webapp is an interactive emulation of the Farfisa Fast 3, a vintage analog organ that can be controlled with a USB-MIDI controller keyboard. The application was made possible with the Web Audio and Web MIDI api's. The Farfisa Fast 3 is a 1960's era analog transistor organ that is known for it's iconic vintage sound found on the records of artist such as Kraftwerk, Jackson 5, The Ventures, and more!
Source: github.com/ryanjamesmcgill/webaudio-organ
Live Demo: farfisaofthefuture.com

Moog Rogue Expansion Module

This is an expansion module I built for the Moog Rogue analog monophonic synthesizer. I found my self using oscillator 2 to create chords with the instrument, but this was difficult since you can only adjust the interval with the a potentiometer. This module allows you to "save" 4 preset intervals for the second oscillator (plus a glide feature!). Please see video for details.
